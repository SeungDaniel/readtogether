# 요한복음 봇 사용설명서 (사용자용)

## 할 수 있는 것
- **공동체 모드**: 단톡방에 매일 “오늘의 본문” 자동 발송 (관리자가 서버/cron으로 설정).
- **1:1 모드**: 개인 DM에서 내 진도로 본문 받기, 진행 상황 확인하기.

## 개인 DM에서 사용 방법
1. 텔레그램에서 봇(@letsreadjohn)을 열고 아래 명령을 입력합니다.
2. 명령 목록
   - `/start` : 시작 안내(명령 설명)
   - `/start_john` : 개인 퀘스트 시작(첫 실행 필수). 이미 진행 중이면 현재 일차 안내.
   - `/next` : 내 다음 퀘스트(일차) 본문 받기 + 자동으로 다음 일차로 진도 올림.
   - `/status` : 완료한 일차/다음 일차 정보 확인.
   - `/repeat` : 직전에 받았던 일차 다시 보기.
   - `/previous` : 직전보다 더 이전 일차 보기(current_day - 2 기준).
   - `/today_group` : 공동체 “오늘” 본문 보기(첫 번째 그룹 설정 기준).
3. 응답이 없다면
   - 봇 서버(폴링 스크립트)가 꺼져 있을 수 있습니다. 운영자에게 “bot_polling.py 실행 여부”를 확인 요청하세요.

## 공동체 모드 (단톡방)
- 매일 지정 시간에 “오늘의 범위/요약/참고자료” 메시지가 방에 자동 발송됩니다.
- 그룹별 시작일/플랜/타임존은 관리자 설정에 따릅니다.

## 자주 묻는 질문
- **/start 입력했는데 아무 반응이 없어요**  
  개인 모드는 `/start_john`을 사용합니다. 그래도 안 나오면 봇 서버가 꺼져 있을 수 있으니 운영자에게 문의하세요.
- **다른 일차를 받고 싶어요**  
  `/next`를 누를 때마다 자동으로 다음 일차로 넘어갑니다. 직전 일차는 `/repeat`로 다시 볼 수 있습니다.
- **공동체 오늘 본문만 보고 싶어요**  
  `/today_group` 명령을 사용하세요.

## 참고
- 이 봇은 웹훅이 아니라 **롱폴링 + cron** 방식으로 동작합니다. 운영자가 서버에서 스크립트를 실행해야 합니다.
